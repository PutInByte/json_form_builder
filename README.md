# json_form_builder

Powerful Form Builder for Flutter

### Install Plugin 

1. You need to add this line of code to Wang's pubspec.yaml


```dependencies:
  json_form_builder:
    git:
      url: https://github.com/warioddly/json_form_builder.git
      ref: main
```

2. Next you need to do the following in the terminal

``` 
  flutter pub get
```
